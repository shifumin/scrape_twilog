Twilog scraping
===============
Twilogのツイートをスクレイピングするやつ。  
ある月のツイート一覧(リプライを除く)をスクレイピングする。  
現状@shifuminのツイートしかスクレイピングできない。  


##使い方
例 : 2014年4月のツイートをスクレイピングする。

`$ ruby twilog_scraping.rb`  
`何年？  例:2014`  
`$ 2014`  
`何年？  例:2014`  
`$ 04`  



##今後追加したい機能
* スクレイピングするIDを標準入力から取得する(@shifumin以外でもスクレイピング可能にする)
