Twilog scraping
===============
Twilogのツイートをスクレイピングするやつ。  
特定のTwitterIDのある月のツイート一覧(リプライを除く)をスクレイピングする。  


##使い方
例 : @shifuminの2014年4月のツイートをスクレイピングする。

`$ ruby twilog_scraping.rb`  
`TwitterIDは？  例:shifumin`  
`$ shifumin`  
`何年？  例:2014`  
`$ 2014`  
`何年？  例:2014`  
`$ 04`
